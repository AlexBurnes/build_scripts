## Abstract

Collection of bash scripts used for check and build C++ projects.
Run tools: cmake, conan, cppcheck, clang-format others.

## Version

0.1.0

# Usage

Add as submodule

    git submodule init
    git submodule add https://github.com:AlexBurnes/build_scripts.git scripts
